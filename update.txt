Чтобы добавить IP или подсети в белый список, с которых может авторизоваться админ, в конфигурационный

======= С 0.6.5 до 0.7 версии =======
Внимание! В версии 0.7 основные настройки панели храняться в файле application/config/gameap_config.php а не в config.php,
Чтобы ваши текущие настройки не затерлись, перенесите их в файл gameap_config.php:
----------------------------------
1. Откройте application/config/config.php
----------------------------------
2. Всё, что между 
	/*
	 * GameAP configuration
	*/

и
	/* End of file config.php */

перенесите в файл gameap_config.php
а в это место добавьте 
	include_once 'application/config/gameap_config.php';
----------------------------------	
[обязательно]		Импортируйте SQL файл db/update_to07.sql в базу данных.
[обязательно]		В конфигурационном файле application/config/config.php параметру $config['enable_hooks'] задайте значение TRUE.
[не обязательно] 	Добавьте параметр $config['local_steamcmd_path'], и укажите в нем путь к SteamCMD на локальной машине.
